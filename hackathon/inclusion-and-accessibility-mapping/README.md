# Accessibility and Inclusion Taxonomy Mapping

This file has an mapping of major accessibility and inclusion considerations for volunteering 
activities and organisations. It is intended to provide a more comprehensive list of accessibility
considerations for organisations when advertising volunteering opportunities.

Images have been generated using the [WebVOWL](https://github.com/VisualDataWeb/WebVOWL) ontology visualisation tool.
